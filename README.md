# Eastern Empire Website Clone

A pixel-perfect clone of the Eastern Empire website (https://www.easternempire.com.au/) with enhanced features and full-stack functionality.

## Overview

Eastern Empire is Sydney's premier South Asian live music ensemble. This website showcases their work, allows fans to discover upcoming gigs, listen to their music, learn about the band, and book them for events.

## Features Implemented

### Original Website Features
- ✅ Hero section with autoplay video background
- ✅ Image gallery showcasing performances
- ✅ Newsletter subscription form
- ✅ Contact form
- ✅ Band biography section
- ✅ Featured music videos
- ✅ Exact design, layout, and images from original site

### New Enhanced Features (As Requested)
1. **Gigs & Events Page** - Prominently displays upcoming shows
2. **Music Library & Streaming Hub** - Spotify, SoundCloud, YouTube integration
3. **Comprehensive Band Bio** - Full history with member profiles
4. **Professional Services & Booking Page** - Complete booking system

## Tech Stack

- **Frontend**: React 19, Tailwind CSS, Shadcn/ui
- **Backend**: FastAPI, MongoDB
- **Features**: SEO optimized, fully responsive, form validation

## Pages

- Home (/) - Hero, gallery, newsletter, contact, about
- Gigs (/gigs) - Upcoming and past shows
- Music (/music) - Spotify, SoundCloud, YouTube embeds
- Bio (/bio) - Band story, timeline, members
- Bookings (/bookings) - Services and booking form

## Running

Frontend: http://localhost:3000
Backend: http://localhost:8001/api

Restart: `sudo supervisorctl restart all`
